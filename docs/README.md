# Documentation

Reference library for the ACP-ACRS Feeds Dispatcher project. This folder is for things you **read and consult** — not things you run.

## Contents

| Folder | Audience | What's inside |
|--------|----------|---------------|
| [architecture/](architecture/) | Engineers, Architects | System design, ADRs, diagrams |
| [business/](business/) | Product, Leadership | Product brief, value map, stakeholders |
| [onboarding/](onboarding/) | Publishers, Consumers | Step-by-step guides to connect to the dispatcher |
| [runbooks/](runbooks/) | On-call Engineers | Incident response, DLQ investigation, deployment checklist |
| [compliance/](compliance/) | Engineers, Security | ACP Label Framework status and evidence |

## Contribution Guidelines

- Keep docs close to the code they describe — if it changes, update the docs in the same PR
- Use Mermaid for diagrams (renders natively on GitHub)
- ADRs are permanent records — never delete or edit a closed ADR
