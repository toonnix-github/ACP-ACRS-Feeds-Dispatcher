# Feedback

A place for async feedback from anyone involved in the project — consumers, publishers, team members, stakeholders.

## How to Submit Feedback

Create a new file in this folder named: `YYYY-MM-DD-short-topic.md`

Example: `2026-04-15-consumer-presigned-url-ttl.md`

Include:
- **Who**: your team / name
- **What**: the feedback, question, or suggestion
- **Impact**: why it matters
- **Suggested action**: optional — what you think should happen

## Or Use GitHub Discussions

For threaded discussion, open a GitHub Discussion in this repo instead of a file.

---

*Feedback here is reviewed during sprint planning and triaged into the backlog if actionable.*
